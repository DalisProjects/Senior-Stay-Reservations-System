ElderTravel Reservation System
Overview

ElderTravel is a full-stack web application designed to facilitate travel and accommodation for the elderly. It allows visitors to book stays, partners to list their properties, and admins to manage the entire platform.

Features

Visitor Accounts: Users can create accounts to browse and make reservations.

Partner Accounts: Property owners can sign up to list their accommodations, manage bookings, and update property details.

Admin Panel: Admins can oversee all user accounts, approve partner listings, manage reservations, and view statistics.

Installation
ANGULAR VERSION REQUIRED: 14.2.13
Clone the repository:
git clone <repository-url>

Install dependencies in main folder and also for backend:
npm install
(Install all packages needed for the backend server.js libraries)
Configure environment variables:
Create a .env file based on the provided .env.example.

Run the application:
npm start

Usage

Visitors: Sign up to create an account, browse available properties, and make reservations.

Partners: Register as a partner to list properties, manage bookings, and update property details.

Admins: Access the admin panel to approve partner listings, manage users, and monitor platform statistics.
